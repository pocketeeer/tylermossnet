tyler.exe: A Discord Bot

My friends and I like to use Discord to talk online, just like millions of other internet users. Discord is a fun, easy to use 'chatroom'-like software that allows for 'servers' to be made. Along with these servers, you can add bots to your server. These bots can be for moderation, for fun, again, whatever you'd like them to do. Some can play music, some can edit images, some can do in-chat trivia. My friends like to add a lot of bots to do a lot of different things when it comes to our own server. With the server being 60+ people, it can be confusing with all the commands and prefixes to remember when wanting to use the bots. So I decided to make my own bot that does it all.
