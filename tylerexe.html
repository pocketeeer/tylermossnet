tyler.exe: A Discord Bot

tyler.exe is a custom made Discord bot for a private server of 60+ members. The bot includes a fully-customizable user profile system, moderation commands + logs system, daily birthday announcements, a polls system, fun user actions, and more. The goal of this project was to understand how to write software whose aim is to interact with many people on a daily basis. For a project like this, it was important to focus on creating a mainly text command-driven system that is easy to interact with, yet can still effectively deliver what the user wants/needs. The following paragraph is on the background of this project and where it originated.

My friends and I like to use Discord to talk online, just like millions of other internet users. Discord is a fun, easy to use 'chatroom'-like software that allows for 'servers' to be made. Each server is unique, and can be customized into an online environment for chatting about whatever you like; games, shows, movies, people, fashion â€” there's a discord server for everything. Along with these servers, you can add bots to your server. These bots can be for moderation, for fun, again, whatever you'd like them to do. Some can play music, some can edit images, some can do in-chat trivia. My friends like to add a lot of bots to do a lot of different things when it comes to our own server. With the server being 60+ people, it can be confusing with all the commands and prefixes to remember when wanting to use the bots. So I decided to make my own bot that does it all using discord.py!

There are multiple categories of commands to choose from. Here's a breakdown of each category and it's commands!

SERVER
  • info: get detailed info about the server.
  • setlogs: set the logs channel.
  • jail: view members who are currently banned from the server.
  • boosters: view members who are currently boosting the server.
  • ping: view the current latency of the bot.
MODERATION
  • kick: kick the mentioned user.
  • ban: ban the mentioned user.
  • unban: unban the mentioned user.
  • clear: clear x amount of above messages.
PROFILES
  • profile: view your profile (or the mentioned user's profile).
  • editprofile: edit sections of your profile.
  • createprofile: create your profile.
QUOTES
  • quote: attach an image and mention a user to add the image to the user's quotes profile.
  • quotes: view your quotes (or the mentioned user's quotes).
  • delquote: delete a quote from a user's profile.
ACTIONS
  • hug: send a virtual hug to the mentioned user.
  • kiss: send a virtual kiss to the mentioned user.
  • marry: propose to the mentioned user!
  • divorce: divorce the mentioned user.
  • slap: send a virtual slap to the mentioned user.
MISC
  • zodiacs: get a list of zodiacs and their corresponding date ranges.
  • birthdays/bdays: view a list of server user's birthdays.
  • avatar/pfp: view your Discord avatar (or the mentioned user's avatar).
  • id: view your Discord account ID (or the mentioned user's ID).
  • fix: someone send a broken image? Use this to try and fix it.

The bot is currently not available for public use in discord servers, as right now it is tailored for a specific server.
